Instructions for running application:

1. Extract rar file onto Desktop or wherever you want to extract

2. Go to extracted folder and open form1.exe located inside dist directory

3. A command prompt will open up. As soon as you see "Running on http://127.0.0.1:5000/", go to your browser and type localhost:5000

4. The app will start on the web browser. To check time and whether the answer was correct or wrong, check the command prompt

6. To quit the app, close the browser tab and press Ctrl+C in the command prompt.
